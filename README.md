# LLM-ассистент для консультаций клиентов в Telegram

## Настройка и запуск

### Предварительные требования

1. Python 3.9 или выше
2. Установленный uv (`pip install uv`)
3. Токен Telegram-бота (см. [инструкцию по созданию бота](doc/guides/botfather.md))

### Установка

1. Клонируйте репозиторий:
```
git clone <url-репозитория>
cd llmstart-homework
```

2. Создайте файл `.env` в корне проекта со следующим содержимым:
```
TELEGRAM_BOT_TOKEN=your_bot_token_here
LOG_LEVEL=INFO
```

3. Установите зависимости:
```
make setup
```

### Запуск

Для запуска бота выполните:
```
make run
```

Для остановки бота нажмите `Ctrl+C`.

## Тестирование

Чтобы проверить работу бота:
1. Отправьте команду `/start`
2. Отправьте любое текстовое сообщение
3. Бот должен ответить тем же сообщением (эхо)
